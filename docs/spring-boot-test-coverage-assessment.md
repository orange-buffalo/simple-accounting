# Spring Boot Test Coverage Assessment

This inventory includes Spring Boot tests that extend `SaIntegrationTestBase`, excluding full stack tests that extend
`SaFullStackTestBase` and GraphQL API tests under `business/api` or `infra/graphql`. The two document upload/download
REST endpoint tests under `business/api/documents` are included because they are not GraphQL API tests.

The mock/spy bean column tracks Spring test bean overrides defined in the test class, such as `@MockitoBean`,
`@MockBean`, `@SpyBean`, or `@MockitoSpyBean`. The legacy security column tracks deprecated mock security annotations
from `SecurityMocks.kt`, including `@WithSaMockUser` and the `@WithMock*User` shortcuts.

`@WithMock*User` and `@WithSaMockUser` only populate Spring Security context. They do not create database users. The
tests that use them either create the matching users through `preconditions` / `lazyPreconditions` / `EntitiesFactory`,
or do not require a persisted user for the asserted path. Those annotations are therefore tracked only in the legacy
security column, not as database manipulation outside the preconditions API.
